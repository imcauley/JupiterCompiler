# Jupiter Compiler
A J-- Compiler

## Compilation 
`cd src`
`make compiler`

The program 'compiler' will be put in the main directory

## Running
./compiler filename

## Current Limitations
- Cannot send error when there is a while out of a while loop
- Errors do not have line numbers
